This directory includes a few sample CNN Model to classify crops condition from 5 crops.

*   Source datsets can be found on kaggle with the link below:
    [five-crop-diseases-dataset](https://www.kaggle.com/shubham2703/five-crop-diseases-dataset)
    This dataset contain 13,324 images that seperate into 5 crops with each crops identify by healthy and disease condition

*   CNN model's built using CNN algorithm using Convo2D, and the result showen the model has 
    87% accuracy

*   `anscombe.json` contains a copy of
    [Anscombe's quartet](https://en.wikipedia.org/wiki/Anscombe%27s_quartet); it
    was originally described in

    the model saved into three seperate ways **TFJS**, **TF-Lite**, **SavedModel**
